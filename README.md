# 📌 Features Implemented


## Setup & Familiarization

* Forked and cloned the Nexus repository
* Configured and ran the project locally
* Explored component architecture and project structure
* Improved UI consistency:

  * Typography
  * Color palette
  * Responsive layouts
  * Grid alignment


### : Meeting Scheduling Calendar

Integrated a calendar-based scheduling system using React calendar components.

### Features

* Add availability slots
* Modify meeting availability
* Send meeting requests
* Accept or decline meeting requests
* Display confirmed meetings on dashboard


###  Video Calling Section

Implemented a frontend video call interface using WebRTC concepts.

### Features

* Start call button
* End call button
* Video toggle
* Audio mute/unmute
* Screen sharing 


### : Document Processing Chamber

Built a document management chamber for handling contracts and deal-related documents.

### Features

* Upload PDF/documents
* Document preview
* Signature pad mockup
* Status tracking:

  * Draft
  * In Review
  * Signed

# Tech Stack

* React.js
* Next.js
* Tailwind CSS
* WebRTC (UI integration)
* React Calendar / FullCalendar
